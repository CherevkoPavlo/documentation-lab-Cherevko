# Smart Home Climate Monitoring System

Це сервіс моніторингу клімату в розумному будинку, який допомагає
відстежувати температуру, вологість та якість повітря в приміщеннях
у режимі реального часу.

## Основні функції
- Моніторинг температури та вологості
- Перегляд історії показників
- Сповіщення про критичні значення клімату
- Веб-інтерфейс для перегляду даних

## Для кого цей продукт
- Власники розумних будинків
- Користувачі, які піклуються про комфорт і здоров’я
- Початківці, які вивчають IoT та документацію
