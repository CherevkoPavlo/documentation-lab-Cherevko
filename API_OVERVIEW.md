# Огляд API

## GET /api/climate
Отримати поточні показники клімату.

**Запит:**
```json
GET /api/climate
Відповідь:

{
  "temperature": 22,
  "humidity": 45
}
POST /api/climate

Надіслати нові показники з датчика.

Запит:

{
  "temperature": 23,
  "humidity": 50
}


Відповідь:

{
  "status": "success"
}
Коди помилок

200 — успішно

400 — некоректний запит

404 — ресурс не знайдено

500 — помилка сервера
